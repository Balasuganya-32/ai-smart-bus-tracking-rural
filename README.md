# AI Smart Bus Tracking System for Rural Areas 🚌

## Problem
In rural areas, villagers do not know the exact arrival time of buses. 
Students and workers wait for long hours due to lack of real-time information.

## Solution
An AI-powered smart bus tracking system that:
- Tracks buses using GPS
- Predicts Estimated Time of Arrival (ETA)
- Sends delay alerts
- Improves rural transport efficiency

## Features
- Live Bus Status
- AI-based ETA Prediction
- Simple Web Interface
- Scalable for Government Transport

## Technology Stack
Frontend: HTML, CSS, JavaScript
Backend (Future Scope): Python, Flask
AI Model: ETA Prediction using ML
Maps Integration: Google Maps API

## Impact
- Saves time for students & workers
- Reduces uncertainty in villages
- Supports Digital India mission

## Future Enhancement
- Real GPS integration
- SMS alerts for non-smartphone users
- Traffic-aware AI model
